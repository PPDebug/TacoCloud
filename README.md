# Spring in action

这个项目主要是学习《Spring in action》过程中的笔记和代码记录


![Spring in action](docs/images/book.png)


会在每一章节的学习结束时打上一个tag, 可以方便地利用git版本控制回滚到相应的章节进行查看。
如:
```shell
git tag  # 查看标签
git checkout chapter1  # 切换到chapter1标签对应的版本
```
